# 文件整理大师

自动整理指定目录中的文件，包括分类、重命名、去重和归档。

## 触发条件

当用户提出以下需求时激活此技能：
- "整理下载文件夹" / "整理桌面"
- "给这些文件重命名"
- "查找重复文件"
- "归档旧文件"
- "把某目录的文件按类型分类"

## 工作流程

### 1. 扫描与分类
执行 `scripts/organize.py scan <目录路径>` 获取文件清单及分类建议。
- 按扩展名归类（文档、图片、视频、音频、压缩包、代码、其他）
- 显示每个类别的文件数量和占用空间

### 2. 重命名
执行 `scripts/organize.py rename <目录路径> --pattern "<模式>"` 批量重命名。
- 支持模式：`{date}` `{index}` `{ext}` `{original}`
- 默认先预览变更，用户确认后再执行

### 3. 查找重复文件
执行 `scripts/find_duplicates.py <目录路径>` 查找内容相同的文件。
- 使用 SHA256 哈希比对
- 列出重复组，标记可安全删除的副本

### 4. 归档
执行 `scripts/organize.py archive <目录路径> --days <天数>` 将 N 天前未修改的文件移至归档目录。

## 安全规则
- 任何删除操作必须先列出文件清单并等待用户确认
- 重命名操作默认只预览，加 `--execute` 才真正执行
- 绝不操作系统关键目录（C:\Windows, C:\Program Files 等）
