# 日常自动化任务

处理日常重复性任务的自动化工具集。

## 触发条件

- "备份某个文件夹"
- "设置提醒"
- "生成周报"
- "每天/每周帮我做 X"
- "创建定时任务"

## 工作流程

### 1. 文件夹备份
执行 `scripts/backup.py --source <源目录> --target <备份目录>` 创建增量备份。
- 首次全量复制，后续仅复制变更文件
- 保留最近 N 个版本（默认 7 个）
- 生成备份日志
- 支持排除指定模式的文件（如 `*.tmp`, `node_modules/`）

### 2. 定时提醒
使用内置 `CronCreate` 工具创建定时任务，提醒用户。
- "提醒我下午 3 点开会" → 创建一次性 Cron 任务
- "每天早上 9 点提醒我站会" → 创建重复 Cron 任务
- 提醒消息会出现在 Claude Code 会话中

### 3. 周报生成
执行 `scripts/weekly_report.py --project <项目名> --git-dir <Git目录>` 自动生成周报初稿。
- 从 Git 提交记录提取本周工作摘要
- 支持自定义模板填入项目进度
- 输出 Markdown 格式，方便手动编辑完善

### 4. 文件夹监控（高级）
执行 `scripts/watch_folder.py --path <目录> --action <动作>` 监控目录变化。
- 检测文件新增、修改、删除
- 支持自动触发指定动作（如自动归类新下载文件）

## 依赖
- Python 3.8+
- Git（用于周报功能）
- 备份和监控功能纯 Python 标准库，无额外依赖
